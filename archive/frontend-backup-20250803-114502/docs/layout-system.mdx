# Layout Components

Casino-Club의 레이아웃 시스템은 다음과 같은 컴포넌트들로 구성되어 있습니다:

## 1. AppLayout

메인 레이아웃 컴포넌트입니다. 앱의 전체적인 구조를 정의합니다.

### 주요 기능
- AppBar, Sidebar, BottomNav 통합
- 반응형 디자인 자동 적용
- Safe Area 자동 처리
- 컨테이너 크기 조절
- 메트릭/통계 디스플레이

## 2. GameLayout

게임 화면에 특화된 레이아웃 컴포넌트입니다.

### 주요 기능
- 전체 화면 모드
- 게임 컨트롤 영역
- 점수/상태 표시
- 애니메이션 효과

## 3. AuthLayout

인증/회원가입 화면을 위한 레이아웃입니다.

### 주요 기능
- 중앙 정렬 디자인
- 로고/브랜딩 영역
- 폼 컨테이너
- 백그라운드 효과

## 4. Container

콘텐츠를 감싸는 기본 컨테이너입니다.

### 주요 기능
- 반응형 너비
- 패딩/마진 처리
- 최대 너비 제한
- 중앙 정렬

## 5. Header (AppBar)

상단 헤더/앱바 컴포넌트입니다.

### 주요 기능
- 타이틀 표시
- 네비게이션 컨트롤
- 액션 버튼
- 스크롤 효과

## 6. Sidebar

사이드 메뉴 컴포넌트입니다.

### 주요 기능
- 접기/펼치기
- 네비게이션 메뉴
- 사용자 프로필
- 반응형 동작

## 7. BottomNav

하단 네비게이션 바입니다.

### 주요 기능
- 주요 섹션 이동
- 아이콘+라벨 표시
- 활성 상태 표시
- 모바일 최적화

## 레이아웃 가이드라인

1. **기본 원칙**
   - 8px 그리드 시스템 사용
   - Safe Area 항상 고려
   - 반응형 디자인 우선
   - 접근성 준수

2. **반응형 브레이크포인트**
   ```css
   sm: 640px   // 모바일
   md: 768px   // 태블릿
   lg: 1024px  // 작은 데스크톱
   xl: 1280px  // 큰 데스크톱
   2xl: 1536px // 초대형 화면
   ```

3. **여백 시스템**
   ```css
   --spacing-1: 8px
   --spacing-2: 16px
   --spacing-3: 24px
   --spacing-4: 32px
   --spacing-5: 40px
   --spacing-6: 48px
   ```

4. **컨테이너 크기**
   ```css
   sm: 640px
   md: 768px
   lg: 1024px
   xl: 1280px
   full: 100%
   ```

5. **z-index 레이어**
   ```css
   --z-background: 0
   --z-content: 10
   --z-header: 20
   --z-sidebar: 30
   --z-modal: 40
   --z-tooltip: 50
   ```

## 접근성 가이드라인

1. **키보드 네비게이션**
   - 모든 인터랙티브 요소에 포커스 가능
   - 논리적인 탭 순서
   - 키보드 단축키 지원

2. **ARIA 레이블**
   - 모든 중요 영역에 적절한 role 할당
   - 의미 있는 레이블 제공
   - 상태 변화 알림

3. **반응형 접근성**
   - 모든 화면 크기에서 사용 가능
   - 터치 타겟 크기 준수 (최소 44x44px)
   - 제스처 인터랙션 지원

4. **성능 최적화**
   - 컴포넌트 지연 로딩
   - 애니메이션 최적화
   - 리렌더링 최소화
